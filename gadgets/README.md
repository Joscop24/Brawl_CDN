Gadget icons for brawlers, linked by their IDs from [cards](https://api.brawlify.com/game/csv_logic/cards), [the official API](https://developer.brawlstars.com/), or [BrawlAPI](https://brawlapi.com/).

# Standards

## regular
Gadget icons with the in-game green border. Typically around 245x256. All assets must be compressed.
